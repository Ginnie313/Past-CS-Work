Project Description

Our project outputs information about college majors given user requests. This web app is aimed at people deciding between different majors (i.e, high school students or college students) who would like additional economic information to inform their decision. As of the time of submission, we are unaware of any major bugs in the project.

Project Features

This web app will allow you to view all information for one specific major, view information for all majors at once, and has two search options. Basic search lets you list all majors in a category of your choice, sorted by either unemployment rate or median income. Advanced search additionally let you filter the results by up to 6 more parameters, and allows you to sort by any parameter in either direction.
